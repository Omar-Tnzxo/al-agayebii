🔍 شرح تفصيلي: كيف يؤثر كل حقل على العرض
───────────────────────────────────────────────────────────────

1️⃣ title (العنوان)
   ┌─────────────────────────────────────┐
   │  📦 هنا يظهر العنوان               │ ← العنوان كبير بارز
   └─────────────────────────────────────┘

2️⃣ subtitle (العنوان الفرعي)
   ┌─────────────────────────────────────┐
   │  📦 العنوان الرئيسي                │
   │  هنا يظهر العنوان الفرعي           │ ← نص أصغر رمادي
   └─────────────────────────────────────┘

3️⃣ layout (التخطيط)
   
   🔲 grid (شبكة):
   ┌───┬───┬───┬───┐
   │ 1 │ 2 │ 3 │ 4 │
   ├───┼───┼───┼───┤
   │ 5 │ 6 │ 7 │ 8 │
   └───┴───┴───┴───┘
   
   🎞️ slider (سلايدر):
   [←] ┌───┬───┬───┬───┐ [→]
       │ 1 │ 2 │ 3 │ 4 │
       └───┴───┴───┴───┘
       ● ○ ○ (مؤشرات)

4️⃣ columns (الأعمدة)
   
   columns = 2:
   ┌─────┬─────┐
   │  1  │  2  │
   └─────┴─────┘
   
   columns = 4:
   ┌───┬───┬───┬───┐
   │ 1 │ 2 │ 3 │ 4 │
   └───┴───┴───┴───┘
   
   columns = 6:
   ┌─┬─┬─┬─┬─┬─┐
   │1│2│3│4│5│6│
   └─┴─┴─┴─┴─┴─┘

5️⃣ product_count (عدد المنتجات)
   
   product_count = 4:
   ┌───┬───┬───┬───┐
   │ 1 │ 2 │ 3 │ 4 │
   └───┴───┴───┴───┘
   
   product_count = 8:
   ┌───┬───┬───┬───┐
   │ 1 │ 2 │ 3 │ 4 │
   ├───┼───┼───┼───┤
   │ 5 │ 6 │ 7 │ 8 │
   └───┴───┴───┴───┘

6️⃣ show_view_all (زر عرض الكل)
   
   show_view_all = true:
   ┌─────────────────────────────────────┐
   │  📦 العنوان        [عرض الكل →]   │ ← الزر يظهر
   └─────────────────────────────────────┘
   
   show_view_all = false:
   ┌─────────────────────────────────────┐
   │  📦 العنوان                         │ ← بدون زر
   └─────────────────────────────────────┘

7️⃣ sort_order (الترتيب)
   
   في الصفحة الرئيسية:
   
   sort_order = 10  →  │ القسم الأول   │
   sort_order = 20  →  │ القسم الثاني  │
   sort_order = 30  →  │ القسم الثالث  │

8️⃣ is_active (مفعّل/معطّل)
   
   is_active = true:
   الصفحة الرئيسية:  ✅ القسم يظهر
   
   is_active = false:
   الصفحة الرئيسية:  ❌ القسم مخفي تماماً

9️⃣ product_source (مصدر المنتجات)
   
   best_sellers:  → يجلب الأكثر مبيعاً
   new:          → يجلب الأحدث
   deals:        → يجلب المخفضة فقط
   category:     → يجلب من فئة معينة
   manual:       → يجلب المنتجات المحددة يدوياً
