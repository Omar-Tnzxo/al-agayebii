✅ تم إضافة خاصية إعادة ترتيب الأقسام!

═══════════════════════════════════════════════════════════════
🎯 كيفية الاستخدام
═══════════════════════════════════════════════════════════════

1. افتح صفحة إدارة الأقسام:
   /dashboard/homepage/sections

2. سترى كل قسم مع:
   ⬆️ سهم لأعلى
   ⬇️ سهم لأسفل
   [رقم] رقم الترتيب

3. لنقل قسم:
   • اضغط ⬆️ لتحريكه للأعلى
   • اضغط ⬇️ لتحريكه للأسفل

4. سيتم حفظ الترتيب تلقائياً!


═══════════════════════════════════════════════════════════════
💡 مثال توضيحي
═══════════════════════════════════════════════════════════════

قبل:
┌──────────────────────────────┐
│ ⬆️⬇️ [1] الأكثر مبيعاً        │
│ ⬆️⬇️ [2] وصل حديثاً           │
│ ⬆️⬇️ [3] عروض خاصة            │
└──────────────────────────────┘

بعد الضغط على ⬆️ للقسم "عروض خاصة":
┌──────────────────────────────┐
│ ⬆️⬇️ [1] الأكثر مبيعاً        │
│ ⬆️⬇️ [2] عروض خاصة    ← تحرك  │
│ ⬆️⬇️ [3] وصل حديثاً    ← نزل  │
└──────────────────────────────┘


═══════════════════════════════════════════════════════════════
🎨 التصميم الجديد
═══════════════════════════════════════════════════════════════

كل بطاقة قسم الآن تحتوي على:

┌────────────────────────────────────────────────┐
│ ⬆️  [1] 📦 عنوان القسم   [📋][👁️][✏️][🗑️]     │
│ ⬇️      [منتجات]                              │
│        الترتيب: 10 | المصدر: يدوي             │
└────────────────────────────────────────────────┘

العناصر:
• ⬆️⬇️ = أزرار التحريك
• [1] = رقم الترتيب الحالي
• 📦 = أيقونة نوع القسم
• [📋] = إدارة المنتجات (للأقسام اليدوية)
• [👁️] = تفعيل/تعطيل
• [✏️] = تعديل
• [🗑️] = حذف

⚡ الميزات
═══════════════════════════════════════════════════════════════

✅ تحريك فوري بنقرة واحدة
✅ حفظ تلقائي في قاعدة البيانات
✅ رقم الترتيب يظهر بوضوح
✅ السهم الأول معطّل (لا يمكن تحريكه لأعلى)
✅ السهم الأخير معطّل (لا يمكن تحريكه لأسفل)
✅ تحديث فوري للعرض


═══════════════════════════════════════════════════════════════
🔧 كيف يعمل؟
═══════════════════════════════════════════════════════════════

عند الضغط على ⬆️ أو ⬇️:

1. يتم تبديل sort_order بين القسمين
2. يُحفظ التغيير في قاعدة البيانات
3. يُحدّث العرض تلقائياً
4. الصفحة الرئيسية تعكس الترتيب الجديد
